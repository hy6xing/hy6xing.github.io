# 适用于 PC 的 AIMP v5.11

  • 常规：支持 utf8 编码字符串中的代理项字符
  • 常规：WavPack 编解码器已更新到 v5.6
  • 播放器：网络电台 - 支持 24 位（或更高）流
  • 播放器：互联网电台 - 支持基于 JSON 的元数据
  • 播放列表：支持 “评论” 标签字段
  • 播放列表：Linux - 通过拖放重新排序的项目不起作用

  • 音乐库：按父文件夹对表项目进行分组（为此使用文件名列）
