# 进入设置

系统总开关


输入框里直接输入 广告 ➡️ 选 系统广告 右边按钮关掉。
记住涉及应用以免等会漏掉，并非都要动。

涉及应用

# 天气

天气

打开天气 - 右上角 3 个点里➡️选择设置➡️下拉到用户体验计划➡️关闭右边开关

# 日历

日历

同上操作：打开日历➡️右上角设置➡️用户体验计划➡️关掉开关

# 手机管家

手机管家

打开手机管家 ➡️ 右上角设置➡️下拉到 内容推荐设置 ➡️ 关掉开关。
点击 ➡️ 垃圾清理 下拉到内容推荐设置 ➡️ 关掉开关。

# 下载管理

下载管理

打开下载管理 (在桌面的 == 系统工具 == 文件夹里) ➡️ 右上角设置 ➡️ 信息流设置 3 个全部关掉

# 应用安装器

应用安装器

安装应用的时候如果有广告，就点右上角 ⚙ 设置里关掉推荐

# 应用商店

应用商店

打开应用商店 ➡️右下角我的 ➡️ 右上角⚙设置 ➡️通知设置看你的喜好选择性关掉
同样设置界面下拉→功能设置 关掉福利活动

# 自带浏览器

浏览器

打开浏览器右下角菜单 ➡️⚙ ➡️ 信息通知管理 ➡️ 可全部关掉

同样的⚙设置里面上面的主页设置 选中 简洁版 即可，所有的推送全部关掉
就可以得到下图简洁的浏览器界面

浏览器界面

# 下拉类通知面广告

状态栏下拉

除去系统之外你也会遇到一些 下拉通知栏 （图下图）的第三方应用推送的广告。
每次都要点❌   关键是点完下次还会有。

处理起来也很简单 长按 你要管理的条条 弹出 更多设置 通知类别 运营信息 全部关掉或者选择性关掉。除了运营信息外也还可能有别的 也可以选择性关闭。以后就不会再弹啦😁

# 关闭指定应用通知广告的推送

这里以 快手 为例
桌面 长按快手 app 图标➡️应用信息➡️ 通知管理 ➡️ 通知类别里的推送可以全部关闭，也可以选择性关闭(除去重要外推荐全部关闭)。

# 软件的启动广告

各种软件的启动广告估计各位都不喜欢的吧，要么等几秒钟，要么手动点跳过。
我可是记得以前各个厂家的手机发布会都吹自家的手机打开应用多快多快，看看现在，不提了吧。
任你再牛逼的骁龙 8+++ + 多少都没有用，开屏启动广告欢迎您！
所以我们就需要一款能帮我们自动点击跳过广告的软件，以达到软件秒开 看不见开屏启动广告➡️李跳跳 app。当然也还有别的，这里不一一列举。
这里以 MissLee_派大星 2.01.apk(李跳跳)版本为例。需要到无障碍功能选项。
首先下载安装后打开软件
1. 点击中间开关 ➡️ 按照提示 OK ➡️ 选择 已下载的应用

2. 继续找到 ️️❤️李跳跳・new ➡️上面一行 使用❤️李跳跳・new➡️勾选读秒确定即可。

蓝奏云下载链接：https://hy6xing.lanzouw.com/ieIYG0mzqshe

好了，就先到这里了。光编写这篇文章就用了 2 个多小时，主要是处理图片，手机截图完传电脑再拼接，再上传图床，最后再复制图片链接放 markdown 里。
除了这些或许还有，但你只要不去启动它，它就不会出现。出现也是去设置里关掉推荐就好了。
